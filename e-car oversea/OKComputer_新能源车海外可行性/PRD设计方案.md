# 中国新能源品牌出海平台 - PRD设计方案

## 一、项目概述

### 1.1 项目背景
基于中国新能源汽车出海的巨大市场机会，搭建一个面向海外市场的中国新能源品牌展示与线索获取平台，帮助海外经销商、消费者了解中国新能源汽车品牌，并获取销售线索。

### 1.2 项目目标
- **核心目标**：获取销售线索，连接海外买家与中国新能源品牌
- **用户群体**：海外经销商、企业采购方、个人消费者
- **平台定位**：中国新能源汽车出海的"窗口"和"桥梁"

### 1.3 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                      前端展示网站                            │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐       │
│  │ 品牌展示  │ │ 车型介绍  │ │ 新闻资讯  │ │ 线索获取  │       │
│  └──────────┘ └──────────┘ └──────────┘ └──────────┘       │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼

│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐       │
│  │ 素材管理  │ │ 线索管理  │ │ 数据分析  │ │ 用户管理  │       │
│  └──────────┘ └──────────┘ └──────────┘ └──────────┘       │
└─────────────────────────────────────────────────────────────┘
```

### 3.3 API设计

#### 3.3.1 品牌相关API

```
GET    /api/brands              # 获取品牌列表

DELETE /api/brands/:id          # 删除品牌（需认证）
```

```
GET    /api/models              # 获取车型列表（支持筛选）

PUT    /api/models/:id          # 更新车型（需认证）
DELETE /api/models/:id          # 删除车型（需认证）
```

#### 3.3.3 线索相关API

```
POST   /api/leads               # 提交线索（公开）

GET    /api/leads/:id           # 获取线索详情（需认证）
PUT    /api/leads/:id           # 更新线索（需认证）
DELETE /api/leads/:id           # 删除线索（需认证）
```

#### 3.3.4 素材相关API

```
POST   /api/assets/upload       # 上传素材（需认证）
GET    /api/assets              # 获取素材列表（需认证）
DELETE /api/assets/:id          # 删除素材（需认证）
```

#### 3.3.5 认证相关API


### 4.2 数据表结构

| 字段 | 类型 | 说明 |
