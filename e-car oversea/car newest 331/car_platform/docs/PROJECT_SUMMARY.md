# 汽车信息平台 - 项目总结

## 项目概述

汽车信息平台是一个专业的汽车信息聚合系统，自动从多个主流汽车网站抓取最新车型、价格、资讯等数据，为用户提供全面的汽车信息服务。

## 核心功能

### 1. 内容自动抓取
- **多平台支持**: 汽车之家、懂车帝、易车
- **数据类型**: 车型信息、价格动态、销量排行、汽车资讯
- **抓取频率**: 
  - 资讯: 每30分钟
  - 价格: 每小时
  - 车型: 每天
  - 销量: 每天

### 2. 数据自动更新
- **定时调度**: APScheduler实现自动化任务调度
- **增量更新**: 智能识别新增和变更数据
- **数据清洗**: 自动验证和清洗抓取的数据
- **失败重试**: 自动重试机制，确保数据完整性

### 3. 产品优化（产品经理视角）
- **用户体验**: 响应式设计，支持PC和移动端
- **搜索筛选**: 多维度筛选，快速定位目标车型
- **价格监控**: 实时价格变动追踪
- **销量排行**: 了解市场热门车型
- **资讯聚合**: 一站式获取汽车行业动态

### 4. 全流程测试
- **API测试**: 覆盖所有后端接口
- **爬虫测试**: 验证数据抓取和解析
- **前端测试**: 验证页面渲染和交互
- **集成测试**: 验证端到端流程

## 技术架构

### 后端 (FastAPI)
- **框架**: FastAPI + SQLAlchemy
- **数据库**: PostgreSQL
- **缓存**: Redis
- **API文档**: 自动生成Swagger文档

### 爬虫 (Scrapy)
- **框架**: Scrapy
- **反爬机制**: 
  - IP代理池
  - User-Agent轮换
  - 请求频率控制
  - 自动限速
- **数据管道**: 验证 -> 去重 -> 存储

### 前端 (React)
- **框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式**: Tailwind CSS
- **路由**: React Router

## 核心特性

### 1. 反爬机制
- **代理IP池**: 自动切换代理IP
- **User-Agent轮换**: 模拟不同浏览器
- **请求头伪装**: 模拟真实用户行为
- **访问频率控制**: 自动限速，避免被封

### 2. 数据质量保证
- **数据验证**: 多层级数据校验
- **去重处理**: 基于多种策略去重
- **异常处理**: 完善的错误处理和日志
- **数据备份**: 定期备份机制

### 3. 安全防护
- **HTTPS**: SSL证书加密
- **CORS**: 跨域请求控制
- **速率限制**: 防止恶意请求
- **SQL注入防护**: 参数化查询

### 4. 性能优化
- **数据库索引**: 优化查询性能
- **缓存机制**: Redis缓存热点数据
- **图片优化**: 缩略图生成和懒加载
- **Gzip压缩**: 减少传输大小

## 部署方式

### Docker Compose (推荐)
```bash
docker-compose up -d
```

### 手动部署
```bash
# 后端
cd backend && uvicorn main:app --host 0.0.0.0 --port 8000

# 前端
cd frontend && npm run dev

# 爬虫
cd crawler && python scheduler.py
```
